# Agents.md - Project Overview & AI Context

このファイルは、AI エージェントがリポジトリの全体像を把握し、より正確なドキュメント（README.md等）を生成するためのコンテキストを提供します。

## プロジェクトの目的
`repository-C` は、サブモジュール（`repository-A`）の情報を自動取得し、資料（README.md等）を生成・管理するための専用リポジトリです。

## リポジトリ構造

- **`repository-A/` (Submodule)**: 
    - `app.py`: Flask を使用した API の初期プロトタイプ。
    - API エンドポイントや基本的なルーティングの定義が含まれます。


## AI オートメーション戦略

このプロジェクトでは、サブモジュールの最新情報をもとにAIがREADME.md等の資料を自動生成・更新します。
- **`SKILL.md`**: READMEの出力形式やAIの思考ルールを定義。
- **`Agents.md` (このファイル)**: 設計思想、リポジトリの歴史、AIに汲み取ってほしい「行間」の文脈を記録。

## 開発者へのメッセージ

ソースコードのコメントに書ききれない設計の意図や、資料生成・自動化の運用方針などは、この `Agents.md` に追記してください。AI がその内容を自動的に README に反映させ、常に最新の「プロジェクトの地図」を維持します。
